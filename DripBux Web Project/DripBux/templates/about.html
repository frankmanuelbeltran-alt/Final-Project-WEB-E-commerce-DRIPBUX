About DripBux

DripBux is a Flask-based e-commerce simulation where users buy real clothing like shirts, hoodies, pants, and accessories using a mock Robux currency.

Project Details

  • Realistic checkout process with shipping info
  • Delivery tracking simulation
  • Robux-based payment system
  • Wishlist and review system
  • Admin panel for management
